# Tests whether the molecule is neutral.

The test checks whether all atoms in the molecule have a formal charge
of 0.

## Usage

``` r
is.neutral(mol)
```

## Arguments

- mol:

  The molecule to query. Should be a \`jobjRef\` representing an
  \`IAtomContainer\`

## Value

\`TRUE\` if molecule is neutral, \`FALSE\` otherwise
