# List available descriptor categories

List available descriptor categories

## Usage

``` r
get.desc.categories()
```

## Value

A character vector listing available descriptor categories. This can be
used in
[get.desc.names](https://cdk-r.github.io/cdkr/reference/get.desc.names.md)
