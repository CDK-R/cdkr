# view.table

Create a tabular view of a set of molecules (in 2D) and associated data
columns

## Usage

``` r
view.table(molecules, dat, depictor = NULL)
```

## Arguments

- molecules:

  A list of molecule objects (\`jobjRef\` representing an
  \`IAtomContainer\`)

- dat:

  The `data.frame` associated with the molecules, one per row

- depictor:

  Default `NULL`
