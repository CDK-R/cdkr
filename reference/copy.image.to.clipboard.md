# copy.image.to.clipboard

generate an image and make it available to the system clipboard.

## Usage

``` r
copy.image.to.clipboard(molecule, depictor = NULL)
```

## Arguments

- molecule:

  The molecule to query. Should be a \`jobjRef\` representing an
  \`IAtomContainer\`

- depictor:

  Optional. Default `NULL`. Depictor from `get.depictor`
