# Class cdkFormula, ac class for handling molecular formula

This class handles molecular formulae. It provides extra information
such as the IMolecularFormula Java object, elements contained and number
of them.

## Objects from the Class

Objects can be created using new constructor and filled with a specific
mass and window accuracy
