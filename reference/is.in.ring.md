# is.in.ring

Tests whether an atom is in a ring.

## Usage

``` r
is.in.ring(atom)
```

## Arguments

- atom:

  The atom to test

## Value

\`TRUE\` is the atom is in a ring, \`FALSE\` otherwise

## Details

This assumes that the molecule containing the atom has been
appropriately configured.
