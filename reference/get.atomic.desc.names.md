# Get class names for atomic descriptors

Get class names for atomic descriptors

## Usage

``` r
get.atomic.desc.names(type = "all")
```

## Arguments

- type:

  A string indicating which class of descriptors to return. Specifying
  \`"all"\` will return class names for all molecular descriptors.
  Options include \* topological \* geometrical \* hybrid \*
  constitutional \* protein \* electronic

## Value

A character vector containing class names for atomic descriptors
