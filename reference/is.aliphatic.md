# is.aliphatic

Tests whether an atom is aliphatic.

## Usage

``` r
is.aliphatic(atom)
```

## Arguments

- atom:

  The atom to test

## Value

\`TRUE\` is the atom is aliphatic, \`FALSE\` otherwise

## Details

This assumes that the molecule containing the atom has been
appropriately configured.
