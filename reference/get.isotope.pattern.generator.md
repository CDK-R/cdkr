# Construct an isotope pattern generator.

Constructs an instance of the CDK `IsotopePatternGenerator`, with an
optional minimum abundance specified. This object can be used to
generate all combinatorial chemical isotopes given a structure.

## Usage

``` r
get.isotope.pattern.generator(minAbundance = NULL)
```

## Arguments

- minAbundance:

  The minimum abundance

## Value

A `jobjRef` corresponding to an instance of `IsotopePatternGenerator`
