# Get the atoms from a molecule or bond.

Get the atoms from a molecule or bond.

## Usage

``` r
get.atoms(object)
```

## Arguments

- object:

  A \`jobjRef\` representing either a molecule (\`IAtomContainer\`) or
  bond (\`IBond\`) object.

## Value

A list of \`jobjRef\` representing the \`IAtom\` objects in the molecule
or bond
