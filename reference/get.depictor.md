# get.depictor

return an RcdkDepictor.

## Usage

``` r
get.depictor(
  width = 200,
  height = 200,
  zoom = 1.3,
  style = "cow",
  annotate = "off",
  abbr = "on",
  suppressh = TRUE,
  showTitle = FALSE,
  smaLimit = 100,
  sma = NULL,
  fillToFit = FALSE
)
```

## Arguments

- width:

  Default. `200`

- height:

  Default. `200`

- zoom:

  Default. `1.3`

- style:

  Default. `cow`

- annotate:

  Default. `off`

- abbr:

  Default. `on`

- suppressh:

  Default. `TRUE`

- showTitle:

  Default. `FALSE`

- smaLimit:

  Default. `100`

- sma:

  Default. `NULL`

- fillToFit:

  Defailt. `FALSE`
