# Get the bonds in a molecule.

Get the bonds in a molecule.

## Usage

``` r
get.bonds(mol)
```

## Arguments

- mol:

  A \`jobjRef\` representing the molecule (\`IAtomContainer\`) object.

## Value

A list of \`jobjRef\` representing the bonds (\`IBond\`) objects in the
molecule
