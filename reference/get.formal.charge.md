# get.formal.charge

Get the formal charge on the atom.

## Usage

``` r
get.formal.charge(atom)
```

## Arguments

- atom:

  The atom to query

## Value

An integer representing the formal charge

## Details

By default the formal charge will be 0 (i.e., `NULL` is never returned).
