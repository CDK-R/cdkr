# is.aromatic

Tests whether an atom is aromatic.

## Usage

``` r
is.aromatic(atom)
```

## Arguments

- atom:

  The atom to test

## Value

\`TRUE\` is the atom is aromatic, \`FALSE\` otherwise

## Details

This assumes that the molecule containing the atom has been
appropriately configured.
