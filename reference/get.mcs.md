# get.mcs

get.mcs

## Usage

``` r
get.mcs(mol1, mol2, as.molecule = TRUE)
```

## Arguments

- mol1:

  Required. First molecule to compare. Should be a \`jobjRef\`
  representing an \`IAtomContainer\`

- mol2:

  Required. Second molecule to compare. Should be a \`jobjRef\`
  representing an \`IAtomContainer\`

- as.molecule:

  Optional. Default `TRUE`.
